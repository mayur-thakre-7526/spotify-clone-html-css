# Spotify Clone 🎵

A Spotify-inspired web player UI built using only HTML and CSS.

## Features
- Sidebar navigation
- Sticky navbar
- Music player section
- Responsive design
- Hover effects
- Cards layout

## Technologies Used
- HTML5
- CSS3
- Font Awesome
